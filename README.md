# rotateMatrix

Есть двумерный массив, задание развернуть его на 90 градусов. То есть сделать так, чтобы строки стали столбцами, а столбцы строками.
Пример:

`   [    

[
   [1,2,3],   
   
   [1,1,1],

   
   [1,2,3],   -------> 
   
   [2,2,2], 

   
   [1,2,3],    
   
   [3,3,3],

   
   ]                      ]`
